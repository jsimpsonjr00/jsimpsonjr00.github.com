About {{ brand }}

{{ brand }} is a collaborative community site for sharing shopping deals in real time. Completely managed by its members with the goal of providing up to the minute information on the best the net has to offer in consumer products. All users are equals here and the content reflects the qualities of its members. As such, security and user restrictions are kept to a bare minimum for this technology demonstration project.

The Technology

{{ brand }} is a demonstration project built using Angular JS on the Angular Fire implementation of Firebase. Further Yeoman assists with project setup, development and building while Twitter Bootstrap provides a front end boost.
