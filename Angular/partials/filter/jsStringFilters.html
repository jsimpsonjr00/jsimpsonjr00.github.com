Below are a collection of String Filters used to modify strings prior to reaching the view.

Camel to Capitals

Transforms a camel cased string to a series of first letter capitalized words.

Markup

{{ markup.camelToCaps }}

Result

{{ "jsStringFilters" | camelToCapitals }}

jsCurrency

A custom currency filter built to work with values that include comma seperators.

Markup

{{ markup.jsCurrency }}

Result

{{ "10,000" | jsCurrency }}

Click to See

Implemented to handle situations where currency values for prices were not valid numbers. Chained with the jsCurrency filter an invalid number is replaced with the "Click to See" text.

Markup

{{ markup.clickToSee }}

Result

{{ "undefined" | jsCurrency | clickToSee  }}
